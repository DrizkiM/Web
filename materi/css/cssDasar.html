Cara Kerja CSS

Css bekerja dengan menseleksi element html dengan selector yang bisa diambil melalui beberapa cara yaitu:

  1. Menseleksi berdasarkan Tag (Type Selector)
  2. Menseleksi berdasarkan id atau class
  3. Menseleksi berdasarkan attribute dari suatu element
/*Seleksi berdasarkan tag html*/
p {
 color: grey;
 font-size: 20px;
}
/*Seleksi berdasarkan id atau class*/
#test{ /*berdasarkan id*/
 background-color: yellow;
 margin: 10px;
 padding: 5px;
}
.kotak{ /*berdasarkan class*/
 border: 1px solid black;
 padding: 5px;
}
/*Seleksi type dari suatu element*/
input[type="text"]{
 padding: 20px !important;
}
