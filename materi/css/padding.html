Padding

Padding sama seperti margin,tetapi padding mengatur jarak antara content dengan tepian garis(border) dari suatu element.

Menentukan nilai padding

sama seperti margin,kita dapat menentukan nilai padding berdasarkan sisi dari suatu element:

  • padding-top : nilai
  • padding-right : nilai
  • padding-bottom : nilai
  • padding-left : nilai

cara kedua menggunakan shorthand dari proprty padding : top right bottom left, berdasarkan arah jarum jam.
contoh padding: 10px 5px 15px 20px;

cara ketiga hanya menerapkan value untuk sisi vertikal dan horizontal padding : top&bottom left&right
contoh margin: 0 20px;

Contoh Penerapan

<!DOCTYPE html>
<html>
<head>
<title>margin</title>
<style>
.kotak{
	width: 300px;
	height: 300px;
	margin: 0 auto;
	box-sizing: border-box;
}
.kotak1{
	background-color: lightblue;
	padding: 20px;
}
p{
	margin: 0;
}
</style>
</head>
<body>
<div class="kotak kotak1">
<p>Hello World</p>
</div>
</body>
</html>
