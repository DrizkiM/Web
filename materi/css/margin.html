Margin

Margin merupakan jarak atau ruang terluar pada element yang tidak terlihat,digunakan untuk menjaga jarak antar element satu dengan yang lainnya agar tidak terjadi tabrakan.

Menentukan nilai margin

kita dapat menentukan nilai margin berdasarkan sisi dari suatu element:

  • margin-top : nilai
  • margin-right : nilai
  • margin-bottom : nilai
  • margin-left : nilai

cara kedua menggunakan shorthand dari proprty margin : top right bottom left, berdasarkan arah jarum jam.
contoh margin: 10px 5px 15px 20px;

cara ketiga hanya menerapkan value untuk sisi vertikal dan horizontal margin : top&bottom left&right
contoh margin: 0 20px;

Contoh Penerapan

<!DOCTYPE html>
<html>
<head>
<title>margin</title>
<style>
.kotak{
	width: 150px;
	height: 150px;
}
.kotak1{
	background-color: lightblue;
}
.kotak2{
	background-color: lightgreen;
	margin-top: 50px;
}
</style>
</head>
<body>
<div class="kotak kotak1"></div>
<div class="kotak kotak2"></div>
</body>
</html>
