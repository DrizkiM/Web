Font Styling

Font styling digunakan untuk mengatur font yang akan di tampilkan dalam sebuah halaman website.

  • font-family : digunakan untuk menentukan jenis font yang akan digunakan
  • font-size : menentukan ukuran pada teks
  • font-weight : menentukan ketebalan teks
  • font-style : menerapkan styling pada teks

font-family

Format penulisan

font-family: font-pertama,font-optional,font-optional;

body{
 font-family: Arial,"Times New Roman",Verdana;
}
contoh merubah tipe font menggunakan css,font bertipe 'Times New Roman'

Font-size

Sesuai dengan namanya digunakan untuk mengatur ukuran dari suatu teks

Format penulisan

body{
 font-size: ukuran-font;
 untuk ukuran dapat menggunakan 
 1.px -> ukurannya tetap 
 2.% -> ukurannya dinamis
}

font-weight

font-weight yang digunakan untuk mengatur ketebalan dari font yang ditampilkan. Nilai dari properti ini dapat ditentukan dengan menggunakan numeric values (100 sampai 900) atau dengan menggunakan (normal, bold, bolder, dan lighter).

Format penulisan

body{
 font-weight : 1-900 / (bold,bolder,normal,ligher);
}

font-style

Properti ini digunakan untuk menentukan postur dari teks yang ditampilkan, apakah bentuknya vertikal (normal) atau miring (italic dan oblique).

Format penulisan

body{
 font-style: (normal/italic/oblique);
}

Text Styling

Sama halnya dengan font-styling,text-styling juga digunakan untuk mngatur teks di website.

  • text-align : untuk mengatur tata letak suatu teks
  • text-decoration : untuk mengatur garis di sebuah teks
  • text-transform : mengatur teks bersifat capitalize atau lowecase

text-align

Kita bisa mengatur text alignment pada website seperti kita melakukannya pada aplikasi Microsoft Word dengan menggunakan properti text-align

Format Penulisan

tata-letak{
 text-align: (center/justify/left/right);
}

text-decoration

Properti ini paling sering digunakan ketika kita ingin membuat garis bawah atau underline pada teks.

Format Penulisan

text-decore{
 text-decoration: (none,underline,overline,dan line through)
}

text-transform

Properti yang digunakan untuk merubah kapitalisasi dari suatu teks

Format penulisan

text-trans{
 text-transform: (none,capitalize,uppercase,lowercase);
}
