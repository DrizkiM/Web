CSS Layouting - Float

Float adalah properti css yang akan membuat sebuah element berada di kiri ataupun kanan.

<!DOCTYPE html>
<html>
<head>
<title>Float</title>
<style>
.container{
	margin: 0 auto;
	max-width: 800px;
	/*edit disini*/
	height: 500px;
	border: 2px solid black;
}
.kotak{
	width: 200px;
	height: 200px;
	background-color: lightblue;
	float: right;
}
/*edit disini*/
</style>
</head>
<body>
<div class="container">
<div class="kotak"></div>
<!-- edit disini -->
</div>
</body>
</html>

Float biasa digunakan untuk membuat layouting sebuah website,karena mudah dipahami karena hanya memberi nilai left dan right pada sebuah element.

Masalah pada Float

masalah pada float biasa terjadi karena element container(pembungkus) akan menggangap element di dalamnya yang menerapkan float dianggap tidak ada,terdapat 3 cara untuk mengatasi permasalahan ini

  1. Clear
  2. overflow
  3. micro clearfix

Clear

Untuk menerapkan metode clear dengan cara menambahkan div setelah element yang menerapkan float dengan attribute class clear 

<!DOCTYPE html>
<html>
<head>
<title>Float</title>
<style>
.container{
	margin: 0 auto;
	max-width: 800px;
	/*edit disini*/
	/*height: 500px;*/
	border: 2px solid black;
}
.kotak{
	width: 30%;
	height: 200px;
	background-color: lightblue;
	float: right;
}
/*edit disini*/
.clear{
	clear: right;
}
</style>
</head>
<body>
<div class="container">
<div class="kotak"></div>
<!-- edit disini -->
<div class="clear"></div>
</div>
</body>
</html>

overflow

Untuk menerapkan metode overflow dapat menambahkan property overflow : auto di container(pembungkus) element yang di telah menerapkan float

<!DOCTYPE html>
<html>
<head>
<title>Float</title>
<style>
.container{
	margin: 0 auto;
	max-width: 800px;
	/*edit disini*/
	/*height: 500px;*/
	overflow: auto;
	border: 2px solid black;
}
.kotak{
	width: 30%;
	height: 200px;
	background-color: lightblue;
	float: right;
}
/*edit disini*/
</style>
</head>
<body>
<div class="container">
<div class="kotak"></div>
<!-- edit disini -->
</div>
</body>
</html>

Micro clearfix

Micro clearfix metode yang ditemukan oleh Nicolas Gallagher untuk memperbaiki masalah dalam menerapkan float,dengan metode ini kita hanya cukup memberi attribute class cf di container(pembungkus).

<!DOCTYPE html>
<html>
<head>
<title>Float</title>
<style>
.container{
	margin: 0 auto;
	max-width: 800px;
	/*edit disini*/
	/*height: 500px;*/
	border: 2px solid black;
}
.kotak{
	width: 30%;
	height: 200px;
	background-color: lightblue;
	float: right;
}
/*edit disini*/
.cf:before,
.cf:after {
    content: " "; /* 1 */
    display: table; /* 2 */
}

.cf:after {
    clear: both;
}
</style>
</head>
<body>
<div class="container cf">
<div class="kotak"></div>
<!-- edit disini -->
</div>
</body>
</html>
