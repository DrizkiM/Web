CSS Layouting - Positioning

Property CSSposition digunakan untuk mengatur posisi dari sebuah element html,terdapat 5 value dari properti position yaitu:

  1. static
  2. relative
  3. fixed
  4. absolute
  5. sticky

Sebelum mempelajari positioning ada baiknya memahami bagaimana flow HTML itu sendiri,secara default flow HTMl bersifat static artinya flow normal dengan setiap element bertumpuk ke bawah, sedangkan untuk flow non-static akan keluar dari flow normal HTML.

non-static Flow

Ketika kita membuat suatu element bersifat non-static maka kita akan membuka property baru yaitu: top,left,right dan left yang akan relative terhadap element pembungkus terdekat ataupun dengan browser.

Static

Static adalah value default dari setiap element HTML,yang akan bertumpuk kebawah dan saling berkaitan satu sama lain.

Setiap element yang memiliki posisi static,maka setiap perubahan akan mempengaruhi element lainnya

Relative

Element yang kita posisikan relative diibaratkan keluar dari flow normal dan ketika kita mengatur element tersebut maka tidak akan mempengaruhi elemen sebelum/sesudahnya dengan menggunakan property top,left,right,dan bottom, namun tempat element dibuat masih dianggap ada oleh browser.

Absolute

Element dilihat dari sisi 3D

Sama halnya dengan posisi relative,perbedaan jelas adalah tempat awal element akan diangap tidak ada sehingga akan diisi oleh element selanjutnya untuk posisi absolute akan relatif terhadap element terdekatnya.

Fixed

Seperti posisi absolute,fixed akan dianggap keluar dan akan relatif terhadap jendela browser sehingga ketika kita akan selalu melihat element tersebut di layar jendela browser kita.

</div>
</body>
</html>

Sticky

Posisi Sticky adalah posisi gabungan antara posisi relative dan fixed,dimana posisi akan tetap berada di tempat awalnya namun akan menempel terhadap browser ketika sampai di element tersebut karena itulah posisi ini memiliki sifat gabungan dari relative dan fixed.
