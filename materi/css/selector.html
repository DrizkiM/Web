Selector

Selector di perkenalan CSS telah dijelaskan digunakan untuk menjadi acuan style element tertentu pada html,selector terdapat berbagai macamnya diantara lain:

  1. Selector Tag (Type Selector)
  2. Selector id atau class
  3. Selector berdasarkan attribute dari suatu element
  4. Selector Universal
/*Seleksi berdasarkan tag html*/
p {
 color: grey;
 font-size: 20px;
}
/*Seleksi berdasarkan id atau class*/
#test{ /*berdasarkan id*/
 background-color: yellow;
 margin: 10px;
 padding: 5px;
}
.kotak{ /*berdasarkan class*/
 border: 1px solid black;
 padding: 5px;
}
/*Seleksi type dari suatu element*/
input[type="text"]{
 padding: 20px !important;
}
/*Seleksi type dari suatu element*/
*{
 border-box: box-sizing;
}

Tag Selector

Tag selector menggunakan element html sebagai selectornya,ketika terdapat element yang sama pada html maka akan terkena style yang di terapkan

Class Selector

Class selector menargetkan style berdasarkan attribute class yang ada pada element html,untuk menuliskan intruksi di css menggunakan tanda . lalu di ikuti dengan nama dari classnya

Id Selector

Sama seprti class selector,Id Selector menargetkan style berdasarkan attribute id yang ada pada element html,untuk menuliskan intruksi di css menggunakan tanda # lalu di ikuti dengan nama dari idnya

Pseudo Selector

Pseudo-class

Pseudo-class adalah selector yang menargetkan element yang "tidak terlihat" contohnya seperti sifat element,untuk mengaksesnya menggunakan selector khusus yaitu : setelah kita membuat selector

a:hover{
 color: orange;
}

pseudo-class diatas akan menghasilkan hasil dibawah ini,ketika element a di sorot menggunakan cursor maka akan berubah warna menjadi orange
