Boxmodel

Setiap element html diibaratkan adalah sebuah box atau kotak,seperti kotak pada umumnya pasti terdapat isi,tinggi,lebar,lapisan luar hingga jarak,Namun di Element Html kita menyebutnya padding,border,dan margin

  • Content disini berarti isi dari sebuah element contohnya : text
  • Padding adalah lapisan yang memisahkan antara konten dengan border dari suatu element
  • Border adalah garis yang menyelimuti element tapi untuk tampilan awal border bersifat tidak terlihat atau 0
  • Margin adalah jarak terluar yang memisahkan antara element satu dengan yang lainnya

Secara default sebuah box akan mengikuti ukuran dari sebuah kontent,namun kita dapat mengatur nilai dimensi dari box tersebut dengan properti width dan height untuk kita dapat menggunakan pixel, persentase dalam menentukan dimensi kotak

Overflowing Content

Adakalanya sebuah content akan melebihi panjang ataupun lebar dari sebuah box,untuk mengatasi itu di css terdapat sebuah properti yang bernama overflow,terdapat 4 macam value dari overflow diantaranya:

  • visible : value default dimaan content akan di tampilkan walapun telah melebihi box
  • hidden : value yang akan menyembunyikan content yang telah melebihi box
  • scroll : value yang akan memunculkan scroll di pinggir box baik content melebihi box ataupun tidak
  • auto : value hanya akan memunculkan scroll ketika content melebihi box
