Display

Properti display digunakan untuk mengatur element bagaimana seharusnya ditampilkan,seperti di jelaskan di materi ini bahwa setiap element html secara default tedapat displaynya masing-masing ada yang block dan inline.

Block

setiap element yang memiliki display block akan membuat element selanjutnya berada di bawah element sebelumnya

Element HTML yang memiliki display block

  • div
  • h1-h6
  • p
  • form
  • main
  • header
  • footer

Inline

Element yang memiliki display inline akan membuat element tersebut sejajar dengan elemen yang lain

Element HTML yang memiliki display inline

  • a
  • img
  • span
  • code
  • em
  • dll
