Border

Border adalah garis yang mengelilingi padding beserta content,border biasanya tidak terlihat namun kita dapat mengaturnya dengan property border: width type warna;

Menentukan nilai ketebalan border

sama seperti padding,kita dapat menentukan nilai border berdasarkan sisi dari suatu element:

  • border-top-width : nilai
  • border-right-width : nilai
  • border-bottom-width : nilai
  • border-left-width : nilai

Ataupun dapat menggunakan short-hand dari propertyborder-width: top right bottom left

Menentukan style dari border

Kita dapat menerpakan style dari border dengan border-style: style; ,terdapat beberapa style diantaranya:

  • solid Tipe garis lurus atau tidak terputus - putus)
  • dotted Garis yang dibentuk dari titik-titik
  • dashed Garis yang dibentuk dari serangkaian garis pendek.
  • groove Tipe garis yang berbentuk seperti frame
  • hidden Digunakan untuk menyembunyikan garis pada elemen

Menentukan warna dari border

  • border-top-color : nilai
  • border-right-color : nilai
  • border-bottom-color : nilai
  • border-left-color : nilai

Sama seperti value yang lain kita dapat mensetting warna baik secara individu ataupun mengunakan short-hand border-color: warna-atas warna-kanan warna-bawah warna-kiri
