Penulisan Syntax CSS

Dalam menuliskan syntax CSS terdapat 2 hal yang di perlukan,yaitu

  1. Element yang akan menerapkan style yang biasa di sebut Selector
  2. Intruksi style yang akan di terapkan dalam element tersebut

didalam sebuah intruksi style css dapat menerapkan lebih dari 1 dengan memisahkan antara style dengan tanda ;

p { color : black; }
p adalah selector 
color adalah property style
black adalah value
/*contoh style lebih dari 1*/
p {
 color: black;
 font-size: 20px;
}

Memasukan File Css kedalam html

Terdapat 3 cara untuk memasukan/menuliskan file csss,yaitu :

  1. Inline
  2. Internal
  3. Eksternal

Inline

Inline Css berarti kita memasukan syntax css secara langsung ke dalam element html dengan menggunakan attribute style

<p style="color: blue; font-size: 18px;">
 Hello World
</p>

style diatas mengatur teks menjadi berwarna biru serta mengatur ukuran sebesar 18 px

Internal

Internal Css berarti kita memasukan syntax didalam file html tepatnya di antara tag head dengan menggunakan tag style

<!DOCTYPE html>
<html>
<head>
 <title>Internal Css</title>
<style>
 .box{
  width: 100%;
  height: 400px;
  background-color: grey;
 }
</style>
</head>
<body>
 <div class="box"></div>
</body>
</html>

style diatas mengatur class box menjadi berwarna abu-abu serta mengatur ukuran lebar dari box menjadi 100% dari containernya serta memiliki tinggi 400px

Eksternal

Eksternal Css berarti kita memasukan syntax css yang berbeda folder ke dalam file html dengan menggunakan tag link yang di tulisakan diantara tag head

<!DOCTYPE html>
<html>
<head>
 <title>Eksternal Html</title>
 <link rel="stylesheet" href="folder-tujuan.css">
</head>
<body>
 <div class="kotak"></div>
</body>
</html>

File css yang berada di folder lain

.kotak{
 width: 300px;
 height: 300px;
 border: 1px solid black;
 padding: 5px;
}

style diatas mengatur class kotak dengan ukuran tinggi dan lebar 300px dengan border (garis diluar element) dengan ukuran 1px berwarna hitam lalu mengatur jarak antara konten dengan border suatu element menjadi berukuran 5px

Buatlah file HTML dengan menerapkan salah satu metode memasukan file CSS diatas.
