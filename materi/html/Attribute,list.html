Attribute

Attribute adalah informasi tambahan yang terdapat dalam element html,bisanya setiap attribute memiliki sebuah value(isi) 

<p id="tests" class="judul" style="color: blue; align: center;">
 Paragraph ini berwarna biru
</p >

paragraph ini berwana biru

Element html yang harus memiliki attribute wajib

  1. a atau link,attribute wajib yaitu href
  2. img,attribute wajib yaitu src dan alt
  3. input,attribue wajib type dan name
  4. form,attribute wajibnya action dan method

Link

Link adalah elemen html yang berguna untuk kita pindah ke halaman lain atau ke sebuah situs website. 

< a href="" target="_blank"> ini sebuah link </a>
