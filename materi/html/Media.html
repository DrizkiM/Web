Video

Element Video digunakan untuk menambah video ke dalam html

<!-- syntax dasar untuk tag video -->
<video width="320" height="240" controls poster="">
  <source src="link-ke-video-nya" type="video/tipe-videonya"/>
</video>

untuk atrribute tambahan

  • width : untuk mengatur lebar dari video yang di tampilkan
  • height : untuk mengatur tinggi dari video yang di tampilkan
  • src : attribute terpenting digunakan untuk meng

Ada 2 attribute tambahan untuk tag video

  1. controls/autoplay : mengontrol bagaimana video akan di putar secara otomatis ataupun secara manual
  2. poster : untuk menambah gambar sebelum video di load
