Display Element

Element html terdiri dari 2 display yaitu block,inline serta tambahan inline-block

block

setiap element yang memiliki display block akan membuat element selanjutnya berada di bawah element sebelumnya

Element HTML yang memiliki display block

  • div
  • h1-h6
  • p
  • form
  • header
  • footer

inline

Element yang memiliki display inline akan membuat element tersebut sejajar dengan elemen yang lain

  • a
  • img
  • span
  • code
  • em
  • dll

Inline-block

Element inline-block diartikan suatu element memiliki display campuran antara block dan inline,sifat inline-block adalah menyusun kesamping seperti inline tetapi kita dapat mengatur tingi dan lebar seperti block

Formulir (Form)

Form adalah element html yang digunakan untuk mengumpulkan informasi dari seorang pengguna contohnya adalah saat kita membuat akun di sebuah website maka kita akan menggunakan form sebagai perantara untuk mengumpulkan informasi 

<form action="" method="">
 
 ......
</form>

form memiliki attribute wajib yaitu action dan method

action digunakan untuk mengirim hasil informasi dari pengguna ke tempat tertentu biasanya ke server

method adalah cara pengiriman bisa menggunakan POST ataupun GET

Perbedaan Method POST dan GET

GET
Method GET akan menampilkan informasi di dalam url bar
Method GET digunakan untuk perpindahan halaman dalam HTML
POST
Method POST tidak akan menampilkan informasi di url bar
Method POST digunakan untuk menyimpan informasi yang bersifat privasi

Didalam form kita menggunakan element input untuk mengumpulkan informasi banyak macam tipe input dan digunakan sesuai dengan kebutuhan

  • text
  • radio
  • checkbox
  • password
  • tel
  • number
  • submit
  • dll
