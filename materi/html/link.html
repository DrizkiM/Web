Tag Link

Tag yang berfungsi untuk mengubungkan dokumen HTML dengan file yang berada diluar (eksternal).

  • Link ditempatkan di antara tag head
  • Link biasa digunakan untuk memasukan file .css yang berada di folder lain

Format Penulisan

<link rel="stylesheet" href="file-tujuan.format">
  • rel : digunakan untuk menentukan hubungan antara dokumen asal yang berkaitan dengan file tujuan
  • href : digunakan untuk menentukan lokasi file yang akan di panggil

contoh Penggunaan tag link

<!DOCTYPE html>
<html>
<head>
 <title> testing link </title>
 <link rel="stylesheet" href="style.css">
</head>
<body>
 <div class="box"></div>
</body>
</html>
/*file eksternal style.css*
.box{
 width: 300px;
 height: 300px;
 background-color: lightblue;
}
