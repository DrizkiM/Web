Pengenalan HTML

HTML adalah singkatan dari HyperText Markup Language. Tidak seperti skrip atau bahasa pemrograman yang menggunakan skrip untuk menjalankan fungsi, bahasa markup menggunakan tag untuk mengidentifikasi konten.

File yang dibutuhkan untuk belajar web desain

  1. WEB Server (optional) contoh Xampp,lamp,ataupun mamp
  2. Text editor contoh atom,sublime,notepad++ ataupun visual studio code

Menginstall dan Menjalankan WEB Server

  1. Mendownload dari website resmi apache
  2. download versi terbarunya
  3. lalu install
  4. lalu jalankan web server apache

Struktur WEB

  • HTML : Struktur
  • CSS : Presentasi
  • JS : Perilaku

Strukur untuk membuat HTML


<!DOCTYPE html>
<html>
<head>
 <title></title>
</head>
<body>

</body>
</html>

Struktur dokumen HTML bertipe sandwich,karena bertipe sandwich maka memiliki pembuka dan penutup.

  • < !DOCTYPE html > : digunakan memberitahu web broswer bahwa dokumen yang akan di muat bertipe html
  • < html > : digunakan membuat sebuah dokumen html
  • < head > : berisi informasi tentang web contohnya adalah judul
  • < title > : digunakan judul dari html
  • < body > : struktur utama dalam membuat html,element yang kita buat akan ditampilkan di web browser
