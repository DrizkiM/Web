Viewport

Viewport adalah area yang akan terlihat oleh user baik dari desktop,tablet dan mobile,viewport sendiri digunakan untuk mendesain tampilan menjadi responive artinya tampilan web akan terlihat bagus baik dari tampilan desktip,tablet dan mobile. 

<meta name="viewport" content="width=device-width, initial-scale=1">
  • widht=device-width : digunakan untuk mengatur lebar web mengikuti layar perangkat yang sedang digunakan.
  • intial-scale=1 ; artinya web akan memberikan zoom menjadi normal

Viewport biasanya digunakan dengan css @media

@media screen and ([min-width|max-width: ] [ukuran]){
 ..perintah yang diterapkan..
}

Contoh penggunaan media query beserta viewport

<!DOCTYPE html>
<html>
<head>
 <title>Testing</title>
 <link rel="stylesheet" href="style.css">
</head>
<body>

</body>
</html>

Breakpoint

Breakpoint adalah ruleset CSS untuk memisahkan antara ukuran layar,dibawah ini adalah rulset untuk ukuran web,tablet serta mobile 

@media screen and (min-width: 960px){
	body{ background-color: lightblue; }
}
@media screen and (max-width: 960px){
	body{ background-color: lightgreen; }
}
@media screen and (max-width: 450px){
	body{ background-color: yellow; }
}
