Tag script

Tag yang berfungsi menuliskan script contohnya Javascript ataupun untuk menyambung dengan file yang berada diluar (eksternal).

  • script ditempatkan di antara tag body atupun juga di antara head
  • script biasa digunakan untuk memasukan file .js yang berada di folder lain
  • script biasa digunakan untuk menuliskan script lain di dokumen html

Format Penulisan

<script type="tipe file" src="lokasi file"></script>
  • type : digunakan untuk menentukan tipe file yang akan kita tulis bisa javascript atau bahasa pemrograman lain
  • src : digunakan untuk menentukan lokasi file yang akan di panggil

contoh Penggunaan tag script

<!DOCTYPE html>
<html>
<head>
 <title> testing script </title>
</head>
<body>
 <div class="box"></div>
 <script type="text/javascript">
 let kotak = document.querySelector('.box');
 kotak.addEventListener('click',function(){
  alert('Hello world');
 });
</script>
</body>
</html>
