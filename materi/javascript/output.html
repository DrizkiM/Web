Menampilkan output

Menampilkan output sangat penting dalam javascript untuk melihat hasil dari code yang kita tuliskan,menampilkan output dalam javascript dapat menggunakan 4 cara yaitu:

  • Menggunakan Console
  • Menggunakan dialog pemberitahuan(alert)
  • Menuliskannya sebagai kontent di dalam Element
  • Menuliskan secara langsung di jendela browser

Console

setiap browser telah memiliki console tersendiri,untuk membukanya dengan menggunakan shorcut ctrl+shift+k untuk menampilkan output hasil dari code melalui console dapat menggunakan syntax console.log([yang ingin ditampilkan]). 

let a = 15,b = 5;
let c = a+b;
console.log(c);

Alert

Alert adalah popup,untuk menggunakan dapat menggunakan syntax alert([yang ingin ditampilkan]).

let a = 15,b = 5;
let c = a+b;
alert(c);

innerHTML

Untuk menuliskan hasil kedalam element HTML dapat menggunakan perintah yaitu innerHTML,namun untuk melakukan itu sebelumnya kita harus menseleksi terlebih dahulu element yang akan menampung nilai. 

const testingElement = document.getElementById('writeInElement');
//komentar
function writeInnerhtml() {
 let testingWrite = document.getElementById('testWrite').value;
 testingElement.innerHTML = testingWrite;
}

document.write()

document.write() digunakan untuk menampilkan hasil ke dalam layar browser dan menghapus seluruh element HTML,oleh karena itu document.write() hanya digunakan untuk kepentingan debugging. 

let a = 15,b = 5;
let c = a+b;
document.write('testing');
