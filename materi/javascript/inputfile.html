Penulisan Code Javascript

Untuk menuliskan code Js dapat menggunakan tag script

<script type="text/javascript">
 document.write("Hello World");
</script>

Dimana kita memasukan file Javascript?

  1. Di antar tag head
  2. Di antara tag body -> khususnya di bagian bawah

Rekomendasi penyimpanan file

Penyimpanan file yang paling ideal adalah diantar tag body

Perbedaan peletakan javascript

Diantara tag head

Ketika kita meletakan di antara head,semua function yang telah di tuliskan dalaman kode akan diload terlebih dahulu sebelum content page html,sehingga terkadang beberapa function akan tidak berfungsi.

Diantara tag body

Namun,ketika menempatkan file diantara body maka semua content web page akan di load terlebih dahulu baru script javascript.

Memasukan file Javascript ke HTML

Terdapat 2 cara memasukan/menuliskan file javascript,yaitu:

  • Internal
  • Eskternal

Internal

Internal Javascript berarti kita memasukan syntax didalam file html tepatnya di antara tag head atau body seperti yang dijelaskan materi diatas.

Eksternal

Eksternal Javascript berarti kita memasukan syntax JS yang berbeda folder ke dalam file html dengan menggunakan tag script yang di tuliskan diantara tag head ataupun body

<!DOCTYPE html>
<html>
<head>
	<title>Input File Javascript Eksternal</title>
	<!-- dapat di letakan disini,namun lebih ideal di antara tag body -->
	<script src="file-tujuan.js" type="text/javascript"></script>
</head>
<body>
	
	<script src="file-tujuan.js" type="text/javascript"></script>
</body>
</html>
