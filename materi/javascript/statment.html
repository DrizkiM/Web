Statment

Statment dalam Javascript dapat ditulis dengan 2 tipe yaitu

  • Single-Line Statment
  • Multiple-Line Statment

Serta di setiap akhir statment diakhiri dengan symbol (semicolon) ;

Keyword

Setiap statment dimulai dengan keyword dan setiap keyword memiliki artinya masing-masing,contoh :

  • var
  • let
  • const

Contoh mendeklarasikan variable a

var a = 5;
//keterangan
var adalah keyword untuk membuat variable
a adalah nama variable
5 adalah isi dari variable a
